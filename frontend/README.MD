# react 前端编写

## git使用
1. git clone '远程仓库地址'  
2. cd '本地仓库目录'  
3. npm install  
4. npm run dev  

1. git pull origin '远程仓库-开发分支'  
2. 解决冲突  
3. git add .  
4. git commit -m '修改内容评论'  
5. git push origin '远程仓库-自己的开发分支'   
6. 提交pull request  
7. 通知组长审阅代码后合并分支  

## 目录介绍
* common 公共组件
* components 私有组件
* imges  图片库
* styles 样式库
* pages  页面
